Интегралы

Интеграл — это сумма бесконечно малых величин. Он используется для вычисления площадей, длин, объемов и других величин.

∫ f(x) dx

∫ (f(x) ± g(x)) dx = ∫ f(x) dx ± ∫ g(x) dx

∫ kf(x) dx = k ∫ f(x) dx

∫ dx = x + C

Функция Интеграл
∫ x^n dx 1/(n+1) * x^(n+1) + C
∫ e^x dx e^x + C
∫ sin(x) dx -cos(x) + C
∫ cos(x) dx sin(x) + C

Найти неопределенный интеграл: ∫ (3x² - 2x + 4) dx

∫ (3x² - 2x + 4) dx = (3/3)x³ - (2/2)x² + 4x + C

Ответ: x³ - x² + 4x + C
